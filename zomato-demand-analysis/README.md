# Zomato Data-Driven Decision Analytics

## Overview
This project analyzes the profitability and demand patterns of a food delivery platform using data analytics techniques.

## Tools Used
- Excel
- Data analytics methods
- Monte Carlo Simulation
- CVP Analysis
- n8n workflow automation

## Key Analysis
- Cost Volume Profit (CVP) analysis
- What-if analysis using Goal Seek
- Sensitivity analysis
- Monte Carlo simulation (5000 iterations)

## Insights
- Break-even point around 4200 orders
- Increasing price significantly improves profit
- Demand variability impacts profitability more than pricing changes

## Outcome
The project demonstrates how analytics helps businesses optimize pricing strategies and operational decisions.
